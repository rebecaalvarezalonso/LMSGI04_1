## Ejercicio 1 – CIFP Pau Casesnoves

Se ha modelado la información de alumnos y profesores mediante:

- **DTD** (`cifp.dtd`) XML válido con referencia al DTD (`cifp_dtd.xml`)
- **XSD** con restricciones y enumeraciones (`cifp.xsd`) XML válido con referencia al XSD (`cifp_xsd.xml`)
- **JSON Schema** equivalente (`cifp.schema.json`) Documento JSON válido (`cifp.json`)

## Ejercicio 2 – Sindicación de contenidos

A partir del documento HTML proporcionado se han creado:

- Canal de noticias en formato **RSS 2.0** (`coches.rss.xml`)
- Canal de noticias en formato **Atom 1.0** (`coches.atom.xml`)

## Validaciones

Todos los documentos han sido comprobados mediante validadores XML/JSON y el Feed Validator del W3C para garantizar que son correctos y cumplen el enunciado.
